DSA-2000 Telescope Array

Overview

The DSA-2000 will consist of 2000 x 5m dishes covering the 0.7-2 GHz frequency range, spanning an area of 19 km x 15 km in a radio-quiet valley in Nevada. The traditional correlator digital backend is replaced with a "radio camera" that produces images in real-time. As a radio survey instrument it will be unprecedented relative to any instrument existing or planned.

CGEM Telescope

Overview

CGEM is a joint project between UBC, NASA Goddard Flight Center, and NRC DRAO. It will map polarization across the sky at 10 GHz to help search for gravitational radiation from the early Universe.

Azimuthal Pointing System

System collecting azimuthal pointing data for the telescope using an STM32 H723ZG microcontroller and a Hohner 24-bit absolute encoder.

Inclinometer Pointing System

System collecting inclinometer pointing data for the telescope using an STM32 H723ZG microcontroller and an AD7738 24-bit sigma delta ADC.

Currently debugging spurious/aliased signals being picked up by the ADC.

Mechanical Testing

Machined mechanical setup to validate a 20-bit absolute Hohner Encoder.

RF Noise Validation

Using FieldFox spectrum analyzer to validate RF noise from exposed Encoder in Anechoic chamber.
